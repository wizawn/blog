行行行，你们要的 Windows 教程来了。
我不说太多，直接上干货。

**先说需要啥：**
- Windows 10/11
- Node.js 22
- 2GB 硬盘空间

**安装步骤：**
1. 装 Node.js
去 nodejs.org 下载 LTS 版
一路下一步就行

2. 装 OpenClaw
npm install -g openclaw

3. 运行向导
openclaw onboard

就这三步。

**配置 API：**
编辑 ~/.openclaw/openclaw.json
填你的 API Key
（免费额度够用）

**启动服务：**
openclaw gateway
浏览器打开 localhost:18789
搞定。

**常见问题：**
- 权限错误 → 用管理员运行
- 端口占用 → 换个端口
- 网络超时 → 用淘宝镜像
