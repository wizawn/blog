## 博客修复

### 问题
- 博客本地构建后 public/index.html 为空
- Cloudflare Pages 部署返回 404
- Cloudflare API Token 无效 (9109 Invalid access token)

### 根本原因
Hexo 7.x 的 `load_config.js` 使用 `ctx.render.render()` 读取 YAML 配置文件，但此时渲染器尚未加载，导致 YAML 被当作文本读取而非解析。

### 解决方案
修改 `/root/wizawn/node_modules/hexo/dist/hexo/load_config.js`，使用 `js-yaml` 直接读取配置文件：

```javascript
const fs_1 = __importDefault(require("fs"));
const js_yaml_1 = __importDefault(require("js-yaml"));
// ...
try {
    config = js_yaml_1.default.load(fs_1.default.readFileSync(configPath, 'utf8'));
} catch (e) {
    config = await ctx.render.render({ path: configPath });
}
```
