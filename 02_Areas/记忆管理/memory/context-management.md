# 上下文管理配置

## 自动保存规则

### 定时记忆保存（每 2 小时）
- 读取当日 memory/YYYY-MM-DD.md
- 提取重要事件、决策、学习内容
- 更新 MEMORY.md 长期记忆
- 清理过期临时上下文

### 上下文清理（每 4 小时）
- 检查会话运行时间
- 保存重要对话到 memory/
- 清理无用临时文件
- 更新任务状态列表

## 记忆分类

### 长期记忆 (MEMORY.md)
- 用户偏好和习惯
- 重要决策和教训
- 技能和知识积累
- 关系和信任状态

### 短期记忆 (memory/YYYY-MM-DD.md)
- 当日任务日志
- 临时笔记
- 会话记录

### 临时上下文
- 当前任务状态
- 工具使用记录
- API 配置信息

## 清理阈值

| 项目 | 阈值 | 动作 |
|------|------|------|
| 会话时长 | 4 小时 | 提醒保存 |
| 记忆文件 | 7 天 | 归档到长期记忆 |
| 临时文件 | 24 小时 | 自动清理 |
| 任务状态 | 完成 | 移动到归档 |

## 自动化脚本

### 记忆提取
```bash
# 每日执行
1. 读取 memory/YYYY-MM-DD.md
2. 识别重要内容（标记为⭐的条目）
3. 追加到 MEMORY.md
4. 清空当日文件（保留模板）
```

### 上下文清理
```bash
# 每 4 小时执行
1. 检查会话运行时间
2. 保存未完成的任务状态
3. 清理临时文件（>24h）
4. 更新任务列表
```
