# HEARTBEAT.md - 定时任务配置

## 🔥 任务执行前必做 (底层习惯)

### 每次接收任务后、执行前
1. **查阅 MEMORY.md** - 检查 Agent 团队配置、现有工具、历史经验
2. **检查可用队友** - AI Agent Team + RedTeam Agent + DeepAudit
3. **优先团队协作** - 不要孤军奋战！有队友不用是犯罪！
4. **避免重复造轮子** - 先查现有工具和 Agent，再考虑自己开发

| 记忆位置 | 内容 | 用途 |

## 博客部署记忆流程

### 部署成功后自动执行
1. 更新 `memory/blog-deploy-log.md`
2. 记录部署时间、文章列表、分类统计
3. 备份当前文章到 `memory/blog-backup-YYYYMMDD/`
4. 更新下次发布检查清单

### 发布前自动检查
1. 检查 YAML front matter
2. 确认文章在 `content/posts/` 目录
3. 运行本地备份
4. 验证配置文件
