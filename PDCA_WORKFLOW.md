# PDCA 工作流

**实施日期**: 2026-03-03  
**方法来源**: Deming Cycle (Plan-Do-Check-Act)

---

## 🔄 PDCA 循环

```
     ┌─────────────┐
     │   Plan      │ ← 设定目标和计划
     │  (计划)      │
     └──────┬──────┘
            │
            ▼
     ┌─────────────┐
     │    Do       │ ← 实施计划
     │  (执行)      │
     └──────┬──────┘
            │
            ▼
     ┌─────────────┐
     │   Check     │ ← 检查结果
     │  (检查)      │
     └──────┬──────┘
            │
            ▼
     ┌─────────────┐
     │    Act      │ ← 标准化和改进
     │  (处理)      │
     └──────┬──────┘
            │
            └──────────→ 回到 Plan
```

---

## 📋 各阶段详细说明

### P - Plan (计划)

**频率**: 每日 09:00 UTC  
**产出**: 当日任务清单

**检查清单**:
- [ ] 回顾昨日未完成的任务
- [ ] 确认今日优先级 (P0/P1/P2)
- [ ] 估算时间和资源需求
- [ ] 记录到 `tasks/YYYY-MM-DD.md`

**示例**:
```markdown
## 2026-03-03 任务清单

### P0 - 紧急
- [ ] 修复博客文章截断问题
- [ ] 补充 OpenClaw 部署教程

### P1 - 重要
- [ ] 完成 ARL IP 导入
- [ ] 启动 Nuclei 扫描

### P2 - 常规
- [ ] 知乎文章发布
- [ ] 记忆整理
```

---

### D - Do (执行)

**频率**: 持续  
**原则**: 不问直接做，做完再汇报

**执行规则**:
1. **P0 任务** - 立即执行，每 2 分钟汇报进度
2. **P1 任务** - 按顺序执行，每 10 分钟汇报
3. **P2 任务** - 空闲时执行，完成后汇报

**记录**:
- 执行过程记录到 `memory/YYYY-MM-DD.md`
- 遇到的问题记录到 `02_Areas/系统运维/问题记录.md`

---

### C - Check (检查)

**频率**: 每 2 小时 (心跳检测)  
**触发**: HEARTBEAT.md

**检查清单**:
- [ ] 当前任务进度是否正常
- [ ] 是否有阻塞问题
- [ ] 资源使用是否正常 (内存/CPU/磁盘)
- [ ] 是否需要调整优先级

**汇报格式**:
```markdown
💓 心跳检测 - 任务状态更新

### ✅ 已完成
| 任务 | 状态 |
|------|------|
| 任务 A | ✅ |

### 🔄 进行中
| 任务 | 进度 |
|------|------|
| 任务 B | 50% |

### ⏳ 待执行
| 任务 | 优先级 |
|------|--------|
| 任务 C | P1 |
```

---

### A - Act (处理)

**频率**: 每周日 + 问题发生时  
**目的**: 标准化和改进

**行动类型**:

1. **标准化** - 将成功经验写入文档
   - 新技能 → `03_Resources/技能库/`
   - 新模板 → `03_Resources/模板库/`
   - 新流程 → 更新 `HEARTBEAT.md`

2. **改进** - 修正问题
   - 记录教训到 `MEMORY.md`
   - 更新检查清单
   - 调整工作流

3. **归档** - 清理完成的项目
   - 移动到 `04_Archives/`
   - 更新 `00_PARA_INDEX.md`

---

## 📊 任务优先级定义

| 优先级 | 定义 | 响应时间 | 汇报频率 |
|--------|------|---------|---------|
| **P0** | 紧急且重要 | 立即 | 每 2 分钟 |
| **P1** | 重要不紧急 | 1 小时内 | 每 10 分钟 |
| **P2** | 常规任务 | 今日内 | 完成后 |
