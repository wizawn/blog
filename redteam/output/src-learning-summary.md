# SRC 实战学习笔记

## 2026-02-25 学习总结

### 案例 1: 教育 SRC 通用系统渗透
**攻击链:**
1. 网络空间搜索引擎找通用系统
2. 默认凭证登录 → 学生账号存储 XSS
3. 供应商系统注册 → 参数篡改 (改=1) → 后台未授权
4. 爆破 4 位验证码 → 任意用户密码修改
5. 接口爬取 → 未授权访问 (支付设置)
6. SQL Server 堆叠延迟注入

**关键技巧:**
- 供应商系统往往是突破口
- 参数篡改测试 (register 类参数)
- 接口文档爬取自动化

---

### 案例 2: 学校内网渗透
**攻击链:**
1. 官网收集 VPN 默认凭证
2. Burp 爆破工号 (6 位，前缀 208)
3. VPN 内网 → 学生开发系统 (弱口令)
4. 文件上传黑名单绕过: `jsp:$DATA` (Windows 特性)
5. 免杀马生成 → 哥斯拉 jsp× → jspxs√
6. Mimikatz 抓密码 → 弱口令解密
7. Fscan 扫 MS17-010 → 4 台存在
8. CS 隧道 + MSF 攻击 (360 拦截 payload)

**关键技巧:**
- Windows 文件名特性绕过黑名单
- DNS 上线绕过流量监控
- 免杀马需要多格式测试

### 案例 3: 金融单位供应链 + 社工攻击
**攻击链:**
1. 边界突破无果 → 转向第三方电子学习平台
2. EmailAll 收集邮箱 → 100 封钓鱼邮件
3. 伪造登录页 → 等待提交 → 后台文件上传 getshell
4. 数据库 2w+ 员工信息泄露
5. Flash 水坑攻击 → 运维人员鱼叉邮件
6. DNS 上线 CS (绕过 HTTP 限制)
7. AC 设备绕过: Host 头配置为公司官网域名
8. CVE-2020-1472 (Zerologon) → 域控权限

**关键技巧:**
- 供应链攻击优先级高于边界突破
- 水坑攻击 + 鱼叉邮件组合
- DNS 协议上线绕过 AC 设备
- Host 头白名单绕过

### 案例 4: 市级攻防演练 (景区)
**攻击链:**
1. 靶标无突破 → 旁站扫描
2. 子域名枚举 + 端口扫描
3. 目录索引泄露 (老旧系统)
4. Web 服务接口未授权调用
5. 端口变更陷阱 (8**3 → 18**3)

**关键技巧:**
- 旁站往往比靶标更容易突破
- 老旧系统可能存在于非标准端口
- 接口测试注意端口一致性

### 案例 5: 车企攻防演练
**攻击链:**
1. Shiro 反序列化 → 权限维持 tsh
2. 负载均衡绕过 (hostname 判断，2 台机器)

## 共性攻击模式总结

### 信息收集阶段
1. **网络空间搜索引擎**: 找通用系统、默认凭证
2. **子域名 + 端口扫描**: 非标准端口 (18**3)
3. **官网信息**: VPN 说明、默认账号
4. **邮箱收集**: EmailAll 等工具
5. **旁站/供应商**: 往往比靶标本身更脆弱

### 边界突破
1. **Shiro 反序列化**: 经典入口
2. **文件上传**: 黑名单绕过 (`:$DATA`, 空格，双写)
3. **SQL 注入**: 特别是老旧系统接口
4. **未授权访问**: 接口文档爬取
5. **参数篡改**: register=1 类测试

### 内网渗透
1. **DNS 上线**: 绕过 HTTP 监控
2. **MS17-010**: 仍然有效 (445 端口限制需隧道)
3. **LDAP Admin**: 直接导出域管密码
4. **Zabbix 等运维系统**: 游客登录/弱口令
5. **密码复用**: Wiki/堡垒机/Confluence

### 权限维持
1. **免杀马生成**: 多格式测试 (jsp/jpsx)
2. **tsh/CS**: 注意进程创建限制
3. **负载均衡**: 多次执行命令

### 社工技巧
1. **钓鱼邮件**: 伪造登录页
2. **水坑攻击**: Flash 嵌入
3. **鱼叉邮件**: 针对运维人员
4. **密码修改通知**: 紧迫感制造

## 案例 6: 银行资产攻防演练 (2024)

### 攻击面
- 分行文件上传漏洞
- 公众号编辑器
- 小程序
- 企业微信钓鱼
- 其他辅助系统

### 关键突破
**企业微信钓鱼:**
- 备份正常软件 → 上传后门版本 → 强制用户升级
- 钓到总部技术支持 PC
- 获取大量敏感信息 (服务器、堡垒机登录凭证)

### 启示
- 移动端/公众号/小程序往往防护较弱
- 软件供应链是高危攻击面
- 技术支持人员权限高，是优质目标
