## 📊 技术指标

### RSI（相对强弱指数）

```bash
clawguard analyze BTCUSDT --indicator rsi
```

- **超买区**: RSI > 70
- **超卖区**: RSI < 30
- **中性区**: 30 ≤ RSI ≤ 70

### MACD（移动平均收敛发散）

```bash
clawguard analyze BTCUSDT --indicator macd
```

- **金叉**: 快线上穿慢线（买入信号）
- **死叉**: 快线下穿慢线（卖出信号）

### 布林带

```bash
clawguard analyze BTCUSDT --indicator bollinger
```

- **上轨**: 压力位
- **中轨**: 移动平均线
- **下轨**: 支撑位

## 🎯 网格交易策略

### 什么是网格交易？

网格交易是一种**适合震荡行情**的自动化交易策略：

1. 设定价格区间
2. 将资金分成若干份
3. 价格下跌时买入
4. 价格上涨时卖出
5. 赚取差价利润

### 启动网格交易

```bash
clawguard grid --symbol BTCUSDT --upper 70000 --lower 60000 --grids 10
```

参数说明：
- `--symbol`: 交易对
- `--upper`: 价格上限
- `--lower`: 价格下限
- `--grids`: 网格数量
